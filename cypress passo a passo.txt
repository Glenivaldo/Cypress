O que você vai aprender:
Requisitos: Google Chrome, Node.js LTS, VS Code, Cypress
Criando o projeto: npm init -y
Instalando Cypress: npm install --save-dev cypress
Abrindo o Cypress: npx cypress open
Estrutura do projeto: cypress/e2e, fixtures, support
Configuração base: cypress.config.js
